# Glossary

Terms and conventions *Prose*'s docs and CLI surface. The glossary plugin auto-links these terms on every page in the site, so a passing mention of `Pipeline`, `Source`, or `# fmt: off` becomes a hover-revealed tooltip with the same definition this page carries. The page is the index target the inline auto-links route to.

Conventions in this list:

- The headword is the canonical form *(in mono)*. Aliases listed below the headword carry the same definition through to the same routing target
- An entry with a link routes to the canonical look-up page for the term *(rule pages, primitive pages, reference pages)*
- An entry without a link is glossary-only, with no canonical look-up page beyond this list
